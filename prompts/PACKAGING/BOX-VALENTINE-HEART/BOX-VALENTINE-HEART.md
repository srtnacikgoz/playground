# BOX-VALENTINE-HEART

Elegant heart-shaped gift box with flowing ribbon illustration - Valentine's Day collection.

---

## MAIN PROMPT

```
Professional product photography of elegant square chocolate gift box (21x21x4 cm proportions). Closed box with lift-off lid design. Matte warm cream/off-white base color. PRINTED 2D illustration on box - elegant thin flowing ribbon lines forming heart shape, ribbons in burgundy red and soft rose pink with subtle gradient and transparency, delicate hand-drawn vector style like Pierre Marcolini packaging, smooth curved lines NOT thick brushstrokes. COMPOSITION: heart NOT centered, positioned so that 70% of heart is visible on lid, 30% continues down past the lid seam onto box base side. "Sade Chocolate" printed horizontally in one line, small refined typography, positioned left-center-upper area. Minimal elegant Valentine aesthetic. Shot from 45-degree elevated angle showing where heart crosses lid seam. Soft studio lighting, high-key. Standard backdrop: soft off-white (RGB 250,248,245) with 3% subtle texture. 8k photorealistic.
```
