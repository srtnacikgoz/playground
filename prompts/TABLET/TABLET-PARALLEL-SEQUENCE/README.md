# TABLET-PARALLEL-SEQUENCE-V1

Yatay tekrar dizilimi promptu - 5 identik çikolata bar horizontal array.

## Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `PARALLEL-SEQUENCE.md` | Ana prompt dosyası, kullanım talimatları |
| `PARALLEL-SEQUENCE.json` | Structured veri formatı |
| `README.md` | Bu dosya |

## Hızlı Başlangıç

1. `PARALLEL-SEQUENCE.md` dosyasından prompt'u kopyala
2. Referans görseli ile birlikte AI'a gönder
3. Negative prompt'u da eklemeyi unutma

## Konsept

- **Tip:** Repetition Array
- **Düzen:** 5 identik bar, yatay sıra
- **Aralık:** Eşit spacing
- **Açı:** Her bar aynı hafif açıda
- **Işık:** High-key, soft diffused
- **Arka Plan:** Soft off-white, 3% texture

## Önemli

- `unwrapped bare chocolate bars without packaging` ifadesi kritik
- Negative prompt'taki packaging terimleri diğer AI'ler için önemli
- Referans görsel pattern/renk için gerekli
