## Konsept

- **Tip:** Floating Hero Shot
- **Pozisyon:** Tilted 30°, subtle levitation
- **Kamera:** Slightly elevated front
- **Shadow:** Soft beneath (weightless effect)
- **Işık:** High-key, soft diffused

## SINGLE-STANDING vs UNWRAPPED-HERO

- SINGLE-STANDING: Dik, yerde, low angle → "height/stature"
- UNWRAPPED-HERO: Açılı, floating, elevated → "weightless/elegant"
